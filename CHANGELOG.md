## v1.1.0 (2026-06-24)

### 추가 (Added)
- **산업안전보건표지 출력 탭** 신규 추가 (`safety_sign_tab.py` 모듈로 분리)
  - `assets/safety_signs/` 폴더 자동 스캔 → 금지/경고/지시/안내 4개 카테고리, 표지 43종 자동 표시
  - 새 PNG 파일을 넣으면 코드 수정 없이 자동 반영
  - 파일명 → 한글명 자동 변환 (딕셔너리 관리)
  - 화면 3분할: 표지 선택(체크박스) / 실시간 미리보기 / 출력 규격·수량
  - **표지별 수량(부수) 조절** (− / + 버튼)
  - **선택 표지로 한 장 꽉 채우기** 버튼 (동일 표지 채우기 / 여러 표지 균등 분배)
  - 실시간 출력 매수 표시 (선택 수량 / 칸 수 → 출력 장수)
- 인쇄 규격 3종 추가 (label.kr 아이라벨 실측 규격)
  - 아이라벨 424: 100×70mm, 2열×4행(8칸)
  - 아이라벨 812: 196×125mm, 1열×2행(2칸)
  - 아이라벨 211: 199.1×288mm, 1칸
  - 라벨지 실제 여백·간격·모서리 라운드를 정확히 반영

### 변경 (Changed)
- 산업안전보건표지 출력: PDF → **인쇄용 HTML** 방식으로 통일 (GHS 라벨과 동일)
- 인쇄물에서 표지 하단 한글 캡션 제거 (이미지에 이미 문구 포함)
- 표지 이미지를 칸의 90%까지 채우되 비율 유지(`object-fit: contain`)로 깨짐 방지
- GHS 경고표지 / 산업안전보건표지를 상단 **탭**으로 분리

### 수정 (Fixed)
- 오프라인 번들 EXE에서 `icons.json` 누락 오류 해결
  - 수량 버튼 아이콘 → 텍스트(−/+) 교체
  - PyInstaller spec에 flet 데이터(`collect_data_files`) 번들 추가
